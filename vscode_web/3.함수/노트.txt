함수
- 자주쓰는 실행코드를 블록단위로 묶어 놓은 것
- 기능단위로 재사용하기 위해서...
- 선언할 땐 function이나 arrow function사용함
- 호출 : 정의된 함수는 호출해야 비로서 기능이 실행됨

함수의 종류
1. 선언적 함수 - 이름이 붙여져서 정의된 함수
2. 익명함수 -  이름이 없는 함수 - 콜백함수(중요함), Event
:자체로는 호출 불가
:변수에 익명함수를 대입하거나 특정 이벤트 객체에 대입해서 호출함
3.대입형 함수 : 변수에 익명함수가 대입된 형태 (예: 일급함수, 고차함수)
4.즉시실행함수 : 함수가 자기자신을 정의하자마자 바로 자신을 호출함 - 빈도가 높진 않음 - 뭐지?
